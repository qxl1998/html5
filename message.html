餐饮资讯

TRAVEL INFORMATION

————————————

此文是介绍南昌周边的各种美食。美食这件事是萝卜青菜各有所爱,游客们的选择也各有不同，实在难以取舍，就把各类榜单通通放了上来，请你来南昌自行尝试吧~~

瓦罐煨汤

瓦罐煨汤，这个在南昌是比较经典的，可以说，在南昌的早餐、中餐、晚餐店里面基本上都具备这道瓦罐煨汤，十分的美味，值的我们去品尝，小编在冬季的时候就喜欢喝。

admin 18 / 5 / 20

凤米粉蒸肉

在南昌，米粉蒸肉是每到过年用来招待客人的一道美味，现在做的人比较少了，但是这也是一道比较经典的南昌美食，小编对于这道菜也是很喜欢的，五花肉和米粉经过一些工序后，放置锅里去蒸，吃一口很美味。

admin 18 / 5 / 20

藜蒿炒腊肉

每年到了秋冬季节，或者到了年初的那段时间，藜蒿就已经长的差不多高了，南昌人就喜欢去塘边采取藜蒿，配合腊肉来炒，做出的这道菜就叫做藜蒿炒腊肉，十分的美味。

admin 18 / 5 / 20

白糖糕

白糖糕，虽然我吃的比较少，但是这个在南昌来说，这也是一个比较好吃。南昌人使用优质糯米经过几道工序制作完成。

admin 18 / 5 / 20

南昌米粉

南昌米粉，它具有比较悠久的历史，它可以说在南昌的一些早餐店都可以看到这种特色，每天早上一碗米粉，爽滋滋。

admin 18 / 5 / 20

石头街麻花

石头街麻花是江西省南昌市著名的汉族小吃。选用精白面粉，一级白砂糖，上等清油等优质原料，每生产一百斤麻花，要在料里掺进八斤新鲜鸡蛋黄。

admin 18 / 5 / 20

糊羹

南昌传统的小吃，用薯粉调制的糊羹，里面有很丰富的菜料。一碗好的糊羹通常包括鸡肉、鸡胗、猪血、荸荠、香菇、豆腐等等不同食材，切成小丁一锅熬，说通俗点就是一锅“大杂烩”，但真的非常美味。

admin 18 / 5 / 20

灯芯糕

铁拐李”牌灯芯糕是的传统小吃，属江西省四大名特糕点之一，扰说是按八仙之一“铁拐李”亲授的秘方而制作的。产品以优质糯米、白糖为主要原料，配以30多种名贵药材。外观洁白晶莹，糕条柔软，甜而不腻，香味沁人，形似灯芯，可以点燃。品名还寓有步步“登新高”之意。常食还有顺气、活血、健胃、生津、延年益寿之功效，明、清代曾为贡品。现在是当地人们节日喜庆时馈赠亲友的必备礼品。
